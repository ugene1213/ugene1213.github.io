Yowl

Yowl is a single page web application that is inspired by the many business reviewing websites that are available on the web. Users can create their own accounts and write reviews for the various businesses. The backend utilizes Ruby on Rails while the fron end uses React.js. The database used is PostgreSQL and database queries are cahced to prevent unnecessary queries and to optimize load time. Additionally, I used the Google map API for easy visualization of where these particular businesses are located.
